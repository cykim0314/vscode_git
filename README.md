
# 🧑‍💻 Markdown 종합 실습 예제

## 1️⃣ 제목 (Heading)

```markdown
# H1 제일 작은 아기 글씨 예쁘다
## H2 2살 아기 글씨 너무 귀엽다
### H3 3살 아기 글씨 좀 컷네
#### H4 4살 아기 글씨 많이 컷네?
##### H5 5살 아기 글씨 참 많이 크다
###### H6 6살 아기 글씨 아오 무서워

```

---

## 2️⃣ 텍스트 꾸미기

```markdown
**굵은 글씨**
*기울임*
~~취소선~~
`인라인 코드`
```

예시: **Bold**, *Italic*, ~~취소~~, `code`
찬숭이 예시: **굵게** *기울여서* ~~중간에줄쫙~~ `이거 표시 어케 하니`
